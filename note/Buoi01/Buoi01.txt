12. Thẻ tạo danh sách dấu chấm:
<ul>
  <li></li>
  <li></li>
  <li></li>

---------------------------------------------------------------------------------------------------------------------------
13. Thẻ nhóm các nội dung vào một khu vực: <div>
---------------------------------------------------------------------------------------------------------------------------
14. Căn lề cho text : text-align     font-size : cỡ chữ     font-family : loại chữ     font-weight : bold (in đậm)
---------------------------------------------------------------------------------------------------------------------------
Độ rộng : width
---------------------------------------------------------------------------------------------------------------------------
Độ cao : height
---------------------------------------------------------------------------------------------------------------------------
Màu chữ : color
---------------------------------------------------------------------------------------------------------------------------
Màu background : background-color
---------------------------------------------------------------------------------------------------------------------------
15. Khoảng cách giữa các dòng : line-height
---------------------------------------------------------------------------------------------------------------------------
16. Dòng đầu tiên thụt vào so với lề trái : text-indent
---------------------------------------------------------------------------------------------------------------------------
17. Căn lề trái, phải, trên, dưới : padding - 
---------------------------------------------------------------------------------------------------------------------------
18. Căn lề trái và phải : margin - inline
---------------------------------------------------------------------------------------------------------------------------
19. Tạo các dấu chấm để điền nội dung trong form : 

<p style="border-bottom: 2px dotted black ; margin-inline: 1em;"></p>
---------------------------------------------------------------------------------------------------------------------------
20. Thẻ tạo nút bấm : 

<input type="button" name="button" value="Text hiển thị trên button" style="color: red;"(Màu text)>
---------------------------------------------------------------------------------------------------------------------------
21. Tạo khung cho content : style = 

border : 2px solid black; border-radius : 10px; width: 200px; height: 100px; background-color : red; margin: auto;
---------------------------------------------------------------------------------------------------------------------------
22. Phân chia bố cục trái, phải: float : left;  float : right; 
---------------------------------------------------------------------------------------------------------------------------
23. Phân chia bố cục cả trái và phải : clear : both;
---------------------------------------------------------------------------------------------------------------------------
24. Dấu tab

Chèn 1 khoảng trống: &nbsp;
Chèn 2 khoảng trống: &ensp;
Chèn 4 khoảng trống: &emsp;
